# Pomodoro ⏰

A minimalist 1-hour focus timer inspired by Swiss railway clocks with daily goal setting and session tracking.

## Features

- **Real-time clock** - Shows current local time
- **1-hour focus timer** - Click clock to start/stop
- **Visual countdown** - Red overlay fades clockwise
- **Daily goal setting** - Set daily Pomodoro targets
- **Session tracking** - Record completed sessions
- **Streak counter** - Track consecutive days
- **Tab countdown** - See time remaining in browser tab
- **Session persistence** - Continues after page refresh

## How to Use

1. **Set daily goal** - First visit prompts for daily session target
2. **Click the clock** - Start a 1-hour focus session
3. **Watch progress** - Red overlay fades as time progresses
4. **Track completion** - See "Today: X/Y completed" progress
5. **Build streaks** - Maintain daily focus habits
6. **Click to stop** - End session early if needed
